CRM-система

Тех: HTML, CSS, SCSS, JavaScript, Vue2, Vue CLI, HTTP requests, работа с сервером

В программе представлены три раздела: проекты, задачи и личный профиль.

  • при переходе на страницу необходимо пройти авторизацю (сейчас она настроена на любой паспотр / пароль);
  • данные со страницы проектов и задач автоматически генерируются из родительского файла. Их также можно настроить на генерацию с сервера при наличии необходимых данных;
  • задачи и проекты также можно редактировать и удалять;
  • пользователи генерируются с сайта jsonplaceholder. С данного сервера поступают данные о пользователях и их данные;
  • если данные превышают размер, то у пользователя высвечивается подсказки при наведении на информацию и вёрстка не ломается;
  • можно также добавить новую задачу или проект на отдельной странице;
  • при клике на саму задачу или проект пользователь также может перейти на отдельную страницу для ознакомления;

Калькулятор ипотечного платежа

Тех: HTML, CSS, JavaScipt, PHP, nouislider, cleavejs, wnumb

В проекте представлено несколько ипотечных ставок по различным условиям, которые выводятся из отдельного js файла. Пользователь может выбрать стоимость недвижимости, первый взнос, а также срок кредитования. При изменение сумм, ежемесячный платёж будет автоматически пересчитываться.

При выборе ставки кредитования и суммы, ежемесячный платёж высчитывается автоматически и клиет видит минимальный необходимый платёж из расчёта (min - 15%; max - 90%). Все суммы пользователь может как вводить с клавиатуры, так и перетаскивать бегунок.

При каждом изменении суммы в правом углу будет пересчитываться ежемесячный платёж, сумма кредитования и сумма переплаты. Здесь также добавлена возможность подать заявку, а данные будут отправляться на указанную администратором страницу.

При нажатии на "Оставить заявку" появляется поле для ввода личных данных. Форму нельзя отправить без полностью заполненной формы.
